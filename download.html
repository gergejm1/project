How to install on MacOS:

1. Click the "MacOS" button to download the .dmg file.

2. Once downloaded, double-click the .dmg file to open it.

3. Drag the Inventory Vision app into the Applications folder.

4. Open the app from the Applications folder to start using Inventory Vision.

How to install on Windows:

1. Click the "Windows" button to download the .exe file.

2. After the download is complete, double-click the .exe file to start the installer.

3. Follow the on-screen instructions to install the Inventory Vision app.

4. Once installed, launch the app from the Start menu to get started.
